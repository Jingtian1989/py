Py
=========
**Py** is an advanced python writing skills demonstration library.

Includes
---------

**attribute**

	setattr(), getattr(), delattr()


**decorator**

	syntactic sugar, type checker
    
**directory**

	distinct dict
    
**generator**

	yield, generator expression
    
**iterator**

	__next__(), __iter__()
    
    
**list** 
    
	list comprehesion

**naming**

	python naming style

**profile**

	cProfile, timeit, syntactic sugar


**with**

	with .. as ..
